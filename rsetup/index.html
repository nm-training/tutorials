4. Install R packages

Open RStudio > Go to R console > Copy/paste following codes.

install.packages("devtools");

devtools::install_github("nm-training/rsetup");

5. Solve R Package Install Path Problem

Steps to follow:

  1. Search > Advanced System Settings
  2. Click Environment Variables
  3. Add following based on your system:
Name: R_LIBS_USER

Value: C:\Users\Username\Documents\R\win-library

Note: Set Username same as your actual computer username.
