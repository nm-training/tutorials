  • Download R for Windows

  • Download R for Linux

  • Download R for macOS

2. Install RStudio (v1.4.1717)

After downloading R, please download RStudio from following link.

  • Download RStudio

3. Install Rtools4 (only for Windows)

For Windows users, you need to install Rtools4 to install packages required in training.

  • Download Rtools4

4. Install R packages

Open RStudio > Go to R console > Copy/paste following codes.

install.packages("devtools");

devtools::install_github("nm-training/rsetup");
